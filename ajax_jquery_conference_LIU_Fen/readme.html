/*lien du site*/ https://tp-jquery-ajax.fenliulacombe-webdeveloppement.com/ /*guide utilisation - CRUD*/ /*ajouter une présenation*/ sur la page d'acceuil, on peut trouver en haut à droite le bouton "Ajouter une présentation" en couleur rose pour ajouter une présenation. Une fois cliquer sur le bouton, on trouve le formuaire d'ajout. On peut annuler l'action en cliquant sur le bouton annuler. Une fois les informations sont remplis, on peut envoyer en cliquant sur le bouton "envoyer". Par contre, la validation des données ne sont pas faite faut du temps (ceci n'a pas été demandé dans le TP). /*modifier une présenation*/ Le bouton en couleur violet "modifier" en bas à droite de chaque présenation permet de retrouver le formulaire avec les données et modifier les données dans la base de données. /*supprimer une présenation*/ Le bouton en couleur violet "supprimer" en bas à droite de chaque présenation permet de supprimer les données et modifier les données dans la base de données. /*afficher une présenation*/ Toutes les données sont affichés une fois cliquer sur le lien en haut. /*Effet JQuery*/ Les accordions fonctionnent bien mais la fonction sortable() fonctionnement partiellement, donc je l'ai mis en commentaire.
